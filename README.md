# look.sh

From a csv file,
<pre>
Country,Area(sk)
France,643801
Germany,357376
Spain,505990
Italy,301338
</pre>
outputs
<pre>
|                        Country |                       Area(sk) |
| ------------------------------ | ------------------------------ |
|                         France |                         643801 |
|                        Germany |                         357376 |
|                          Spain |                         505990 |
|                          Italy |                         301338 |
</pre>

## Usage
./look.sh file
