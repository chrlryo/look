# look.sh

From a csv file,
<pre>
Country,Area (sk)
France,643 801
Germany,357 376
Spain505 990
Italy,301 338
</pre>
outputs
<pre>
|                        Country |                      Area (sk) |
| ------------------------------ | ------------------------------ |
|                         France |                        643 801 |
|                        Germany |                        357 376 |
|                          Spain |                        505 990 |
|                          Italy |                        301 338 |
</pre>

## Usage
./look.sh file
